# 📊 Smart Data Analytics Platform

An end-to-end Data Analytics application built using **Python, Streamlit, Pandas, and MySQL**. This platform automates data cleaning, KPI generation, dashboard visualization, report generation, and business question answering for uploaded datasets.

---

## 🚀 Features

- 📂 Upload Excel datasets
- 🧹 Automatic ETL (Data Cleaning)
- 📋 Data Quality Report
- 🔍 Automatic Dataset Detection
- 📊 Smart KPI Generation
- 📈 Interactive Dashboard
- 💬 Business Question Answering
- 💾 Save Clean Data to MySQL
- 📄 Export Analysis Report as PDF
- 📗 Export Cleaned Dataset to Excel

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- MySQL

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/glavanya-portfolio/Smart-Data-Analytics-Platform.git
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
streamlit run app.py
```

---

## 📌 Workflow

1. Upload an Excel dataset
2. Perform ETL and data cleaning
3. Generate Data Quality Report
4. Detect dataset type automatically
5. Generate Smart KPIs
6. Visualize data using interactive charts
7. Ask business-related questions
8. Save cleaned data to MySQL
9. Export reports as PDF and Excel
